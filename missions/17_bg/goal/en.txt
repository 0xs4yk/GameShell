Mission goal
============

1/ Run the `xeyes` command, and stop it.
2/ Run the `xeyes` command in the backgroup.


Useful commands
===============

  xeyes
    open a window with 2 eyes that track your mouse

  CMD &
    run the command in the backgroud

  Control-c
    this keybinding usually stop the current command 
    if it is still running
