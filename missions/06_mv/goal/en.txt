Mission goal
============

Pick up all the coins that you can find in the cellar of the castle, 
and place them in your chest (in your cabin in the forest).


Useful commands
===============

mv FILE1 FILE2 ... FILEn DIRECTORY
  move all the files to the directory

~
  abbreviation for the initial directory (its location
  may depend on the shell that is used)
  example : ~/Directory refers to directory "Directory" 
  in the initial directory
