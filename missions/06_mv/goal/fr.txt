Objectif
========

Ramassez toutes les pièces qui se trouvent dans la cave
du château, et placez les dans votre coffre (dans votre
cabane, dans la forêt).


Commandes utilisées
===================

mv FICHIER1 FICHIER2 ... FICHIERn DOSSIER
  déplace les fichiers vers le dossier

~
  abréviation pour le répertoire initial (l'emplacement
  de ce dossier peut dépendre du shell utilisé)
  exemple : ~/Dossier représente le dossier "Dossier" 
  dans le répertoire initial
