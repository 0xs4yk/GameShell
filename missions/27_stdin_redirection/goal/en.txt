Mission goal
============

To get better in the magical art, one needs to know mental math.

Test your multiplication skill!

NOTE: there now is a time constraint.

Hint: the library is rumored to contain some mathematics books and 
hidden volumes...


Useful commands
===============

CMD  <  FILE
  replace the standard input (the keyboard) of the command by a file
  Instead of reading string from the keyboard, the command will read 
  them from the file.
