Objectif
========

Les vieux grimoires de Merlin sont stockés dans son 
bureau, dans la bibliothèque. Vous devez obtenir une 
liste de tous ces grimoires dans un fichier "liste.txt" 
qui devra être stocké dans le tiroir...


Commandes utilisées
===================

  CMD  >  FICHIER
    le résultat de la commande CMD est écrit dans le 
    fichier FICHIER au lieu d'être affiché sur la 
    sortie standard

  less FICHIER
    permet d'afficher un fichier page par page
    Les touches importantes sont
      Espace    pour passer à la page suivante
      q         pour quitter
      /         pour rechercher une chaîne dans le
                fichier
      n         pour passer à l'occurrence suivante de
                la chaîne cherchée
