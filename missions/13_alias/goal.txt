Objectif
========

Créer un alias 'journal' pour pouvoir éditer facilement 
votre fichier journal.


Détails
-------

``nano`` est un éditeur de texte en ligne de commande.
Nous allons l'utiliser pour écrire un petit 
compte-rendu pour le TP...

Le compte-rendu sera un fichier appelé "journal.txt" 
qui devra être sauvegardé dans votre coffre.

Pour éviter de relancer la commande (avec le chemin 
complet du fichier "journal.txt"), nous allons utiliser 
un "alias".



Commandes utilisées
===================

  nano <fichier>
    édite le fichier dans le terminal
    Les raccourcis claviers sont affichés en bas :
      Control-x     pour quitter nano
      Control-o     pour sauver le fichier
      Control-w     pour rechercher dans le texte

  alias NOM='CMD'
    NOM est une chaine de caractère (petite), qui sera 
    remplacé par la commande CMD (longue)
